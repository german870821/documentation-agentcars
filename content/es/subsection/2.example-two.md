## Básico

### Código en línea

Utilice el código en línea para mostrar fragmentos de código dentro de párrafos de texto. Es ideal para referenciar elementos de código directamente en oraciones. `código en línea`

### Bloques de código

Utilice bloques de código para mostrar fragmentos de código de varias líneas con resaltado de sintaxis. Los bloques de código son esenciales para presentar ejemplos de código con claridad.

```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

Al escribir un bloque de código, puede especificar un nombre de archivo que se mostrará sobre él. Se mostrará automáticamente un icono según la extensión o el nombre.
Los nombres de archivo ayudan a los usuarios a comprender la ubicación y el propósito del código dentro de un proyecto.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

Cada bloque de código tiene un botón de copia incorporado que copiará el código a su portapapeles.

::tip{to="https://ui.nuxt.com/getting-started/icons/nuxt#theme"}
Los iconos ya están definidos de forma predeterminada, pero puedes personalizarlos en tu `app.config.ts`:

```ts [app.config.ts]
export default defineAppConfig({
  ui: {
    prose: {
      codeIcon: {
        terminal: 'i-ph-terminal-window-duotone'
      }
    }
  }
})
```

  ```ts [nuxt.config.ts]
    modules: ['@nuxt/ui'],
  
    css: ['~/assets/css/main.css']
  })
  
  ```
  
  ```css [app/assets/css/main.css]
  @import "tailwindcss";
  @import "@nuxt/ui";
  ```

