### Formato de texto

| Estilo | Como usar    | Resultado  |
| ------ | ------------ | ---------- |
| Bold   | `**bold**`   | **Bold**   |
| Italic | `*italic*`   | *Italic*   |
| Strike | `~~strike~~` | ~~Strike~~ |

Combine el formato para obtener estilos de texto más enriquecidos y énfasis visual.

| Estilo        | Como usar           | Resultado         |
| ------------- | ------------------- | ----------------- |
| Bold Italic   | `**_bold italic_**` | ***Bold Italic*** |
| Bold Strike   | `~~**bold**~~`      | ~~**Bold**~~      |
| Italic Strike | `~~*italic*~~`      | ~~*Italic*~~      |

## Listas

Organice elementos relacionados en un formato estructurado y legible. Markdown admite listas desordenadas, ordenadas y anidadas para diversas necesidades de contenido.

### Desordenadas

Utilice listas desordenadas para elementos sin una secuencia específica. Comience cada elemento con un símbolo `-`.

- Item 1.
- Item 2.
- Item 3.

### Ordenadas

Utilice listas ordenadas cuando el orden de los elementos sea importante, como los pasos de un proceso. Comience cada elemento con un número.

1. Item 1.
2. Item 2.
3. Item 3.

### Anidado

Cree listas jerárquicas con subelementos para estructuras complejas. Aplique una sangría de cuatro espacios a los subelementos para anidarlos.

- Item 1.
  - Subitem 1.
  - Subitem 2.
- Item 2.

## Citas en bloque

Resalte citas importantes, citas textuales o texto destacado. Las citas en bloque distinguen visualmente el contenido citado.

### Línea única

Las citas en bloque de una sola línea son ideales para citas breves e impactantes o citas que caben en una sola línea. Para crear una cita en bloque de una sola línea, añada un `>` antes de cada párrafo. Ideal para citas breves e impactantes.

> Nuxt UI es una colección de componentes, elementos componibles y utilidades de Vue, construida sobre Reka UI, orientada a la estructura y el diseño, y diseñada para usarse como bloques de construcción para su aplicación.

### Multilínea

Las citas en bloque multilínea son ideales para citas más largas o cuando se necesitan varios párrafos dentro de una sola cita.

> Nuxt UI es una colección de componentes, elementos componibles y utilidades de Vue, basadas en Reka UI, orientadas a la estructura y el diseño, y diseñadas para usarse como bloques de construcción para tu aplicación.
>
> Crea aplicaciones Vue atractivas, responsivas y accesibles con Nuxt UI.
