Parámetros en las peticiónes
==================================
Se envían los datos básicos de la reserva y adicional la información del vehículo seleccionado

Servicio: `getSelection`
---------------------

| URL                                                   | ENTORNO      |
| ----------------------------------------------------- | ------------ |
| https://api.agentcars.com/v2/sites/get-selection      | producción   |
| https://api.dev.agentcars.com/v2/sites/get-selection  | desarrollo   |


Parámetros de envío
---------------------

Todos los parámetros de envios van por `GET`, parámetros obligatorios vienen de llamada de 'getMatrix'

| VARIABLE         | OBLIGATORIA     | SIGNIFICADO                                                                  |
| -----------------| --------------- | ---------------------------------------------------------------------------- |
| sippCode         |  YES            | Tipo de automóvil según código SIPP                                          |
| ccrc             |  YES            | Viene en la respuesta de la llamada getMatrix                                |
| rateType         |  YES            | Código tipo de Tarifa                                                        |
| cdCode           |  NO             | Código de descuento                                                          |
| pcCode           |  NO             | Código de promoción                                                          |
| companyCode      |  YES            | Código de la empresa que alquila el vehículo                                 |
| country          |  YES            | País de oficina, codigo alfa 2 ej: United States (US), Colombia (CO)         |
| source           |  YES            | País de fuente, codigo alfa 2 ej: United States (US), Colombia (CO)          |
| pickUpDate       |  YES            | Fecha de Pickup - Formato yyyy-mm-dd                                         |
| pickUpHour       |  YES            | Hora de Pickup - Formato militar ej: 0800 -> 8:00 am                         |
| dropOffDate      |  YES            | Dropoff Date - Format yyyy-mm-dd                                             |
| dropOffHour      |  YES            | Hora de Dropoff - Formato militar ej: 0800 -> 8:00 am                        |
| pickUpLocation   |  YES            | Pickup City * - based on `IATA` code of destination airport                  |
| dropOffLocation  |  YES            | Dropoff City * - based on `IATA` code of return airport                      |
| pickUpAddress    |  YES            | Nombre de la oficina de pickup                                               |
| dropOffAddress   |  YES            | Nombre de la oficina de dropoff                                              |
| language         |  NO             | ISO de idioma para recibir respuesta. *                                      |
| paymentType      |  NO             | Tipo de pago del resultado mostrado (ppd: Pagar Ahora, pod: Pago en Destino) |

*Lenguajes disponibles: Inglés (en), Español (es), Japonés (ja), Coreano (ko), Portugués (pt), Alemán (de), Francés (fr), Italiano (it), Árabe (ar)
