Parámetros en la solicitud
============================

Servicio para solicitar la activación de una reserva pendiente por activar, en estado 'ON HOLD'

Servicio: `release`
---------------------

| URL                                                 | ENTORNO         |
| --------------------------------------------------- | --------------- |
| https://api.agentcars.com/v2/sites/release          | producción      |
| https://api.dev.agentcars.com/v2/sites/release      | desarrollo      |

Parámetros de envío
---------------------

Todos los parámetros de envío van por `POST`, a continuación la explicación de cada uno de los parámetros:

| VARIABLE         | OBLIGATORIO  | DESCRIPCIÓN                            |
| ---------------- | ------------ | -------------------------------------- |
| lastName         | SI           | Apellido cliente                       |
| referenceCode    | SI           | Código de referencia de la reserva     |

Ejemplo:
```
  lastName:test
  referenceCode:123123
```

La definición de la respuesta del servicio se encuentra [aqui](2.response.md).
