Tarifas - Petición `REST`
=========================

Servicio que entrega todas las tarifas validas para una fuente - destino.

Servicio:
---------
| URL                                           | ENTORNO       |
| --------------------------------------------- | ------------- |
| https://api.agentcars.com/v2/sites/rates      | producción    |
| https://api.dev.agentcars.com/v2/sites/rates  | desarrollo    |


Párametros:
-----------

Todos los parámetros son enviados por `GET`, A continuación se definen los parámetros:

| VARIABLE  | OBLIGATORIO   | DESCRIPCIÓN                               |
| --------- | ------------- | ----------------------------------------- |
| country   | SI            | Código de país de destino                 |
| source    | SI            | Código de país fuente                     |
| language  | NO            | ISO de idioma para recibir respuesta. *   |

*Lenguajes disponibles: Inglés (en), Español (es), Japonés (ja), Coreano (ko), Portugués (pt), Alemán (de), Francés (fr), Italiano (it), Árabe (ar)

---

```
    - Ejemplo:
        https://api.dev.agentcars.com/v2/sites/rates?access-token=4824a8dc7626f7b5a5338591abe588a2&country=US&source=CO&language=es
    
    - Parámetros en petición:
        country:US
        source:CO
        language:es
```
