Información de la tarifa - Parámetros en las peticiónes
=======================================================

Calcula el valor de las tarifas según las inclusiones adicionales para el alquiler del vehículo

Servicio: `rateInformation`
---------------------

| URL                                                           | ENTORNO    |
| ------------------------------------------------------------- | ---------- |
| https://api.agentcars.com/v2/sites/get-rate-information       | producción |
| https://api.dev.agentcars.com/v2/sites/get-rate-information   | desarrollo |

Parámetros de envío
-------------------

Todos los parámetros de envío van por `GET`

| VARIABLE    | OBLIGATORIO | MEANING                                                                                                       |
| ----------- | ----------- | ------------------------------------------------------------------------------------------------------------- |
| paymentType | SI          | Tipo de pago de la reserva (ppd= prepagada, pod=pago en destino)                                              |
| rateType    | SI          | Id de la tarifa                                                                                               |
| uniqid      | SI          | Id de identificación de la transacción, entregado en respueseta de [get-selection](/v2/selection/response)    |
| language    | NO          | ISO de idioma para recibir respuesta. *                                                                       |

*Lenguajes disponibles: Inglés (en), Español (es), Japonés (ja), Coreano (ko), Portugués (pt), Alemán (de), Francés (fr), Italiano (it), Árabe (ar)

Ejemplo request vía `GET`

```
https://api.dev.agentcars.com/v2/sites/get-rate-information?access-token=72163f79ad5e45256e448e94e6a106bd&paymentType=pod&rateType=1&uniqid=58e7b24f7e5a2&language=es

//Parameters sent in the above string:
    paymentType=pod
    rateType=1
    uniqid=58e7b24f7e5a2
    language=es
    
```
