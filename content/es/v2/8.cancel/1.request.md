Parámetros en la solicitud 
============================

Servicio para solicitar la cancelacion de una reserva

Servicio: `cancel`
---------------------

| URL                                                | ENTORNO         |
| -------------------------------------------------- | --------------- |
| https://api.agentcars.com/v2/sites/cancel          | producción      |
| https://api.dev.agentcars.com/v2/sites/cancel      | desarrollo      |

Parámetros de envío
---------------------

Todos los parámetros de envío van por `POST`, a continuación la explicación de cada uno de los parámetros:

| VARIABLE         | OBLIGATORIO  | DESCRIPCIÓN                            |
| ---------------- | ------------ | -------------------------------------- |
| lastName         | SI           | Apellido del cliente                   |
| confirmationCode | SI           | Código de Confirmación de la rentadora |


Ejemplo:
```
  lastName:test
  confirmationCode:G4N410
```
