Tarifas - Respuesta `JSON`
=======================

En la respuesta se reciben los códigos de las tarifas validas, cada una con la siguiente estructura:

```
    {
        "id",
        "name",
        "description",
        "inclusions",
    },
```

### Valores:

| VARIABLE      | DESCRIPCIÓN                                               |
| ------------- | --------------------------------------------------------- |
| id            | Identificador de la tarifa                                |
| name          | Nombre de la tarifa                                       |
| description   | Descripción de la tarifa                                  |
| inclusions    | Arreglo con las inclusiones y su respectiva descripción   |


```
Ejemplo:
    {
        "id": "3",
        "name": "Protección Total",
        "description": "Protección de Colisión o Perdida (CDW/LDW) sin Franquicia, Daños a Terceros, Un Conductor Adicional, Kilometraje Ilimitado, Impuestos y Sobrecargos Obligatorios",
        "inclusions": {
            "mileage": {
                "description": "La mayoría de las categorías de autos no tienen limite en kilometraje.  Las excepciones son las categorías Deportivas y Súper Lujo.  Estas categorías sí tienen limite."
            },
            "surcharges": {
                "description": "Estos son cargos gubernamentales y obligatorios.  Cantidades y cálculos varían por estado y ciudad. Incluyendo impuestos y sobrecargos en su tarifa hace su viaje mas fácil."
            },
            "cdw/ldw": {
                "description": "Incluyendo la opción Protección de Colisión o Pérdida, usted no será responsable por eventuales daños al auto, siempre y cuando dichos siniestros o daños no sean resultado del uso indebido del vehículo."
            },
            "lis/lsi": {
                "description": "Protección contra terceros (LIS/SLI) brinda hasta $1.000.000 USD de cobertura ante reclamaciones por daños físicos o daños materiales en su contra por parte de terceros perjudicados en caso de accidente. Esta protección cubre su póliza de seguro y/o sus bienes personales por el primer $1.000.000 de dólares americanos (USD) en caso de reclamaciones en su contra."
            }
        }
    }
```
