Parámetros en las peticiónes
=======================================

Servicio que busca todas las oficinas disponibles para una coordenada o un código de ciudad

Servicio: `findOffices`
-------

| URL                                                 | ENVIRONMENT |
| --------------------------------------------------- | ----------- |
| https://api.agentcars.com/v2/sites/find-offices     | production  |
| https://api.dev.agentcars.com/v2/sites/find-offices | development |


Parámetros de envío
---------------------

Todos los parámetros de envíos van por `GET`

| VARIABLE      | OBLIGATORIA   | SIGNIFICADO                                                                                       |
| ------------- |---------------| ------------------------------------------------------------------------------------------------- |
| distance      | YES           | Distancia en millas a buscar oficinas alrededor de la ubicación solicitada                        |
| source        | YES           | País de oficina, codigo alfa 2 ej: United States (US), Colombia (CO)                              |
| lat           | YES*          | Latitud                                                                                           |
| lng           | YES*          | Longitud                                                                                          |
| cityCode      | YES**         | Código de la ciudad (ej: MIA para Miami)                                                          |
| companyCode   | NO            | Código de la empresa que alquila el vehículo (Si quiere buscar solo por una compañia específica)  |

 \* Solo aplica obligatorio si envía coordenadas.
 
 ** Solo aplica obligatorio si no envía coordenadas.

```
    - Ejemplo con coordenadas:
        https://api.dev.agentcars.com/v2/sites/find-offices?access-token=989e82a2c47050b63bc4dff9de7c05fe&distance=2&source=CO&lat=45.463689&lng=9.1883
    
        Variables en petición:
            distance:2
            source:CO
            lat:45.463689
            lng:9.1883
            
    - Ejemplo con código de ciudad:
        https://api.dev.agentcars.com/v2/sites/find-offices?access-token=989e82a2c47050b63bc4dff9de7c05fe&distance=2&source=CO&cityCode=MIA
    
        Variables en petición:
            distance:2
            source:CO
            cityCode;MIA
```

Este servicio se llama para mostrar oficinas, por ejemplo en un mapa, según los parámetros enviados. 

Después se llama una vez el servicio getMatrix con los parámetros de envío correspondientes, si se uso coordenadas llamar este servicio enviando esos parámetros. La lista de oficinas obtenidos puede no corresponder con la lista de carros de getMatrix, por diferentes motivos como horarios, fechas, entre otros.

El detalle de parámetros de envío lo encuentran aca: [get-matrix](/v2/matrix/request)
