Explicación de cada parámetro:

| VARIABLE                     | IMPORTANCIA | DESCRIPCIÓN                                                                  |
|------------------------------|-------------|----------------------------------------------------------------------------- |
| confirmationCode             | `Alta`      | Código de confirmación de la reserva, con la rentadora                       |
| firstName                    | `Alta`      | Nombre cliente                                                               |
| lastName                     | `Alta`      | Apellido cliente                                                             |
| carDescription               | `Alta`      | descripción del vehículo                                                     |
| sippCode                     | `Alta`      | Código sipp del tipo de vehículo                                             |
| carModel                     | `Alta`      | Módelo del vehículo                                                          |
| rateCode                     | `Alta`      | Código de la tarifa                                                          |
| rateTittle                   | `Alta`      | Nombre de la tarifca                                                         |
| rateDescription              | `Alta`      | Descripción de la tarifa                                                     |
| companyImage                 | `Alta`      | Imagen de rentadora                                                          |
| companyName                  | `Alta`      | Nombre de rentadora                                                          |
| days                         | `Alta`      | Días de renta                                                                |
| carImage                     | `Alta`      | Imagen del vehículo                                                          |
| currency                     | `Alta`      | Moneda de transacción                                                        |
| type                         | `Alta`      | prepago (ppd) o pago en destino (pod)                                        |
| voucherAmount                | `Muy alta`  | Monto que debe mostrarse en el voucher como "Voucher Amount". Este campo es exclusivo de las reservas prepagadas "PPD"                                               |
| voucherNumber                | `Muy alta`  | Número que debe mostrarse en el voucher como "Voucher Number". Este campo es exclusivo de las reservas prepagadas "PPD"                                              |
| rateAmount                   | `Alta`      | valor de la reserva                                                          |
| taxesAmount                  | `Alta`      | Valor de impuestos                                                           |
| doors                        | `Alta`      | Número de puertas del vehículo                                               |
| passengers                   | `Alta`      | Capacidad máxima de pasajeros del vehículo                                   |
| bags                         | `Alta`      | Cantidad de maletas que soporta el vehículo                                  |
| airConditioners              | `Alta`      | Aire Acondicionado (Yes/No)                                                  |
| transmission                 | `Alta`      | Tipo de transmisión (Manual / Automatic)                                     |
| pickUpLocation               | `Alta`      | Código IATA de sitio de recogida del vehículo                                |
| dropOffLocation              | `Alta`      | Código IATA de sitio de retorno del vehículo                                 |
| pickUpAddress                | `Alta`      | Dirección de sitio de recogida del vehículo                                  |
| dropOffAddress               | `Alta`      | Dirección de sitio de retorno del vehículo                                   |
| pickUpDate                   | `Alta`      | Fecha recogida - Format yyyy-mm-dd                                           |
| dropOffDate                  | `Alta`      | Fecha retorno - Format yyyy-mm-dd                                            |
| pickUpHour                   | `Alta`      | Hora recogida - Format militar time example: 0800 -> 8:00 am                 |
| dropOffHour                  | `Alta`      | Hora retorno - Format militar time example: 4 pm => 1600                     |
| status                       | `Alta`      | Estado de la transacción *                                                   |
| flight_number                | `Alta`      | Número de vuelo                                                              |
| fFlyerNbr                    | `Alta`      | Número de viajero frecuente                                                  |
| fFlyerCarrier                | `Alta`      | Código de aerolinea                                                          |
| fmember                      | `Alta`      | Número de membresía                                                          |
| convertedCurrency            | `Alta`      | Denomicación de moneda a la que que se convierten los valores a moneda definida por la agencia. **   |
| convertedrateAmount          | `Alta`      | Valor de la reserva convertido a moneda definida por la agencia. **          |
| convertedTaxesAmount         | `Alta`      | Valor de impuestos convertidos a moneda definida por la agencia. **          |
| agency_settlement_currency   | `Alta`      | Moneda con nosotros para la comisión. **                                     |
| agency_commission_settlement | `Alta`      | Comisión de la agencia con nosotros. **                                      |
| voucherInformation           | `Muy alta`  | Información del voucher - Arreglo con la información dividida en 3 categorías (importante, estándar, adicional). Es el campo que se debe mostrar en el voucher que se genera al cliente, junto con el currency que aparecen en la respuesta del confirmation |
| rateInfo                     | `Muy alta`  | Información de la tarifa, el nombre, descripción y las inclusiones que tiene. Más información [aquí](/v2/selection/response#explicación-del-contenido-de-cada-una-de-las-posiciónes).                                                                      |

*Para el proceso a seguir cuando este campo viene con el valor "On Request", por favor mirar la sección [Reservas con Status On Request](reservations-onRequest.md).

** Este campo solo es retornado cuando la agencia contacta con agentcars y solicita que las respuestas del API retornen los montos en una moneda específica. 


#### voucherInformation

Arreglo con la información dividida en 4 categorías explicadas a continuación:

| LLAVE                 | VALOR     |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| important             | Información importante, obligatoria para cargar en el voucher que se debe entregar al cliente, puede traer título, descripción, imagen e información adicional (accountNumber, iata, HCCNumber)
| standard              | Información estándar, obligatoria para cargar en el voucher que se debe entregar al cliente, puede traer título, descripción e imagen |
| additional            | Información adicional, opcional, puede traer título, descripción e imagen                                                             |
| cancellationPolicy    | Información de políticas de cancelación, opcional                                                                                     |

Mayor información acerca de Voucher [aquí](/v2/certification-process/info#vouchers-y-confirmaciones).
