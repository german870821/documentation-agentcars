Confirmación de la reserva - Parámetros en las peticiónes
=========================================================

Se encarga de realizar y confirmar la reserva, segun los datos del vehículo, la tarifa y la informacion requerida para realizarla.

Servicio: `confirmation`
------------------------

| URL                                                   | ENTORNO       |
| ----------------------------------------------------- | ------------- |
| https://api.agentcars.com/v2/sites/confirmation       | producción    |
| http://api.dev.agentcars.com/v2/sites/confirmation    | desarrollo    |

Todos los parámetros de envío van por `POST`

| CAMPO          | OBLIGATORIA|  VALOR                                      | SIGNIFICADO                                                                       | 
| -------------- | ---------- | ------------------------------------------- | --------------------------------------------------------------------------------- | 
| uniqid         | YES        | 58e7a97b506d4                               | Identificador de sesión de la transacción en proceso.                             | 
| paymentType    | YES        | pod                                         | Tipo de pago de la tarifa pod: si es pago en destino,  ppd:prepagada              | 
| rateType       | YES        | 3                                           | Indice de la tarifa seleccionada                                                  | 
| companyCode    | YES        | ZT                                          | Código de la compañía rentadora                                                   | 
| sippCode       | YES        | CCAR                                        | Tipo de vehículo según código SIPP                                                | 
| pickUpLocation | YES        | LAX                                         | Código de ciudad donde llega el cliente por el vehículo                           | 
| dropOffLocation| YES        | LAX                                         | Código de ciudad donde el cliente regresa el vehículo                             | 
| pickUpDate     | YES        | 2022-10-22                                  | Fecha de Pickup                                                                   | 
| dropOffDate    | YES        | 2022-10-24                                  | Fecha de Dropoff                                                                  | 
| pickUpHour     | YES        | 1200                                        | Hora de Pickup en formato militar ej: 4 pm => 1600                                | 
| dropOffHour    | YES        | 1200                                        | Hora de Dropoff en formato militar ej: 4 pm => 1600                               | 
| pickUpAddress  | YES        | LAX, Los Angeles International Airport...   | Oficina donde recoge el vehículo                                                  | 
| dropOffAddress | YES        | LAX, Los Angeles International Airport...   | Oficina donde regresa el vehículo                                                 | 
| firstName      | YES        | test                                        | Nombre del cliente                                                                | 
| lastName       | YES        | test                                        | Apellidos del cliente                                                             | 
| age            | YES        | 1                                           | Rango de edad el cliente segun las siguientes reglas : 1 - Más de 25 años, 2 - Entre 21 y 24 años, 3 - Menos de 21 años   | 
| email          | YES        | test@test.com                               | Email del cliente                                                                 | 
| realBase       | YES        | 118.00                                      | Valor base de la renta (sin impuestos)                                            | 
| realTax        | YES        | 20.00                                       | Valor de los impuestos                                                            | 
| total          | YES        | 118.00                                      | Valor total de la renta (con impuestos)                                           | 
| currency       | YES        | USD                                         | Currency                                                                          | 
| cdCode         | NO         | CDO12                                       | Código Descuento                                                                  | 
| pcCode         | NO         | PC1212                                      | Código Promoción                                                                  | 
| cbs            | NO         | 1/null                                      | Child booster seat, asiento para niños 2-5 años, 1 para indicar que se quiere, null para no   | 
| cst            | NO         | 1/null                                      | Child toddler seat, asiento para niños 0-2 años, 1 para indicar que se quiere, null para no   | 
| gps            | NO         | 1/null                                      | GPS, SOLO si en la tarifa no no se incluye, (no todos los vehiculos tienen esta opcion valida), 1 para indicar que se quiere, null para no    | 
| skyracks       | NO         | 1/null                                      | Sky Racks, SOLO para los lugares validos, de lo contrario regresa error (INVALID SPECIAL EQUIPMENT CODE-SKR), 1 para indicar que se quiere, null para no  | 
| language       | NO         | es/en                                       | ISO de idioma para recibir respuesta. *                                           | 
| flight_number  | NO         | AA1234                                      | Número de vuelo                                                                   | 
| fFlyerNbr      | NO         | ZXY987                                      | Número de viajero frecuente                                                       | 
| fFlyerCarrier  | NO         | A3                                          | Código de aerolinea, descargue listado de aerolineas [aquí](https://api.dev.agentcars.com/airlines.csv)| 
| fmember        | NO         | 0123456789                                  | Número de membresía, solo para las siguientes compañias: Avis, Hertz y National   | 
| on_hold        | NO         | 1                                           | Código para indicar que la reserva tiene que colocarse en estado 'On Hold'        |

*Lenguajes disponibles: Inglés (en), Español (es), Japonés (ja), Coreano (ko), Portugués (pt), Alemán (de), Francés (fr), Italiano (it), Árabe (ar)
