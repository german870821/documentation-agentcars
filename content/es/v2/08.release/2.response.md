Explanation of each parameter:

| VARIABLE                      | MEANING                                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| confirmationCode              | Código de confirmación de la reserva, con la rentadora                                                                    |
| firstName                     | Nombre cliente                                                                                                            |
| lastName                      | Apellido cliente                                                                                                          |
| carDescription                | descripción del vehículo                                                                                                  |
| sippCode                      | Código sipp del tipo de vehículo                                                                                          |
| carModel                      | Módelo del vehículo                                                                                                       |
| rateCode                      | Código de la tarifa                                                                                                       |
| rateTittle                    | Nombre de la tarifca                                                                                                      |
| rateDescription               | Descripción de la tarifa                                                                                                  |
| companyImage                  | Imagen de rentadora                                                                                                       |
| companyName                   | Nombre de rentadora                                                                                                       |
| days                          | Días de renta                                                                                                             |
| carImage                      | Imagen del vehículo                                                                                                       |
| currency                      | Moneda de transacción                                                                                                     |
| type                          | prepago (ppd) o pago en destino (pod)                                                                                     |
| voucherAmount                 | Monto que debe mostrarse en el voucher como "Voucher Amount". Este campo es exclusivo de las reservas prepagadas "PPD"    |
| voucherNumber                 | Número que debe mostrarse en el voucher como "Voucher Number". Este campo es exclusivo de las reservas prepagadas "PPD"   |
| rateAmount                    | valor de la reserva                                                                                                       |
| taxesAmount                   | Valor de impuestos                                                                                                        |
| doors                         | Número de puertas del vehículo                                                                                            |
| passengers                    | Capacidad máxima de pasajeros del vehículo                                                                                |
| bags                          | Cantidad de maletas que soporta el vehículo                                                                               |
| airConditioners               | Aire Acondicionado (Yes/No)                                                                                               |
| transmission                  | Tipo de transmisión (Manual / Automatic)                                                                                  |
| pickUpLocation                | Código IATA de sitio de recogida del vehículo                                                                             |
| dropOffLocation               | Código IATA de sitio de retorno del vehículo                                                                              |
| pickUpAddress                 | Dirección de sitio de recogida del vehículo                                                                               |
| dropOffAddress                | Dirección de sitio de retorno del vehículo                                                                                |
| pickUpDate                    | Fecha recogida - Format yyyy-mm-dd                                                                                        |
| dropOffDate                   | Fecha retorno - Format yyyy-mm-dd                                                                                         |
| pickUpHour                    | Hora recogida - Format militar time example: 0800 -> 8:00 am                                                              |
| dropOffHour                   | Hora retorno - Format militar time example: 4 pm => 1600                                                                  |
| status                        | Estado de la transacción                                                                                                  |
| convertedCurrency             | Denomicación de moneda a la que que se convierten los valores a moneda definida por la agencia. **                        |
| convertedrateAmount           | Valor de la reserva convertido a moneda definida por la agencia. **                                                       |
| convertedTaxesAmount          | Valor de impuestos convertidos a moneda definida por la agencia. **                                                       |
| agency_settlement_currency    | Moneda con nosotros para la comisión. **                                                                                  |
| agency_commission_settlement  | Comisión de la agencia con nosotros. **                                                                                   |
| voucherInformation            | Información del voucher - Arreglo con la información dividida en 3 categorías (importante, estándar, adicional)           |
| rateInfo                      | Información de la tarifa, el nombre, descripción y las inclusiones que tiene. Más información [aquí](/v2/selection/response#explicación-del-contenido-de-cada-una-de-las-posiciónes). |

** Este campo solo es retornado cuando la agencia contacta con agentcars y solicita que las respuestas del API retornen los montos en una moneda específica. 

#### voucherInformation

Arreglo con la información dividida en 4 categorías explicadas a continuación:

| LLAVE                 | VALOR                                                                                                                                 | 
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | 
| important             | Información importante, obligatoria para cargar en el voucher que se debe entregar al cliente, puede traer título, descripción, imagen e información adicional (accountNumber, iata, HCCNumber)
| standard              | Información estándar, obligatoria para cargar en el voucher que se debe entregar al cliente, puede traer título, descripción e imagen |
| additional            | Información adicional, opcional, puede traer título, descripción e imagen                                                             |
| cancellationPolicy    | Información de políticas de cancelación, opcional                                                                                     |

Mayor información acerca de Voucher [aquí](/v2/certification-process/info#vouchers-y-confirmaciones).
