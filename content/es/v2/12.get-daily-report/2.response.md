Respuesta del servidor
===============

Se recibe una matriz separada con todas las reservas diarias del usuario (o agencia si el usuario es su supervisor o administrador):

```
        {
            "confirmationCode",
            "firstName",
            "lastName",
            "email",
            "pickUpDate",
            "dropOffDate",
            "date",
            "status",
            "paymentType"
        },
```
