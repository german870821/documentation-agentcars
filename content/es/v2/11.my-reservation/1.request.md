Parámetros en las peticiónes
============================

Consulta los datos de una reserva

URL del Servicio: `my-reservation`
---------------------

| URL                                                     | ENTORNO     |
| ------------------------------------------------------- | ----------- |
| https://api.agentcars.com/v2/sites/my-reservation       | producción  |
| https://api.dev.agentcars.com/v2/sites/my-reservation   | desarrollo  |

Parámetros de envío
---------------------

Todos los parámetros de envío van por `POST`, a continuación la explicación de cada uno de los parámetros:

| VARIABLE         | OBLIGATORIO  | DESCRIPCIÓN                             |
| ---------------- | ------------ | --------------------------------------- |
| lastName         | SI           | Apellido cliente                        |
| confirmationCode | SI           | Código de Confirmación de la rentadora  |
| language         | NO           | ISO de idioma para recibir respuesta. * |

*Lenguajes disponibles: Inglés (en), Español (es), Japonés (ja), Coreano (ko), Portugués (pt), Alemán (de), Francés (fr), Italiano (it), Árabe (ar)

Ejemplo:
```
  lastName:test
  confirmationCode:G4N410
  language:es
```
