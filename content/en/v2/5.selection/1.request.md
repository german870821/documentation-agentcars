Service Parameters
==================

Basic reservation data sent and additional information on the selected vehicle

Service: `getSelection`
-----------------------

| URL                                                   | ENVIRONMENT  |
| ----------------------------------------------------- | ------------ |
| https://api.agentcars.com/v2/sites/get-selection      | production   |
| https://api.dev.agentcars.com/v2/sites/get-selection  | development  |


Request Parameters
--------------------

All params are sent with `GET`. Mandatory params come in response to the call getMatrix

| VARIABLE         | MANDATORY      | MEANING                                                                     |
| -----------------| -------------- | --------------------------------------------------------------------------- |
| sippCode         | YES            | Type of vehicle according SIPP code                                         |
| ccrc             | YES            | It comes in response to the call getMatrix                                  |
| rateType         | YES            | Type of rate                                                                |
| cdCode           | NO             | Discount Code                                                               |
| pcCode           | NO             | Promotion Code                                                              |
| companyCode      | YES            | Company Code from the chosen vehicle                                        |
| country          | YES            | Country Office * country code alpha-2: United States (US), Colombia (CO)    |
| source           | YES            | Source Country * country code alpha-2: United States (US), Colombia (CO)    |
| pickUpDate       | YES            | Pickup Date - Format yyyy-mm-dd                                             |
| pickUpHour       | YES            | Pickup Hour - Format militar time example: 0800 -> 8:00 am                  |
| dropOffDate      | YES            | Dropoff Date - Format yyyy-mm-dd                                            |
| dropOffHour      | YES            | Dropoff Hour - Format militar time example: 0800 -> 8:00 am                 |
| pickUpLocation   | YES            | Pickup City * - based on `IATA` code of destination airport                 |
| dropOffLocation  | YES            | Dropoff City * - based on `IATA` code of return airport                     |
| pickUpAddress    | YES            | Name of pickup office - It comes in response to the call getMatrix          |
| dropOffAddress   | YES            | Name of dropoff office - It comes in response to the call getMatrix         |
| language         | NO             | ISO To set language response. *                                             |
| paymentType      | NO             | Payment Type of the result shown (ppd: Pay Now, pod: Pay at Destination)    |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

```
https://api.dev.agentcars.com/v2/sites/get-selection?access-token=989e82a2c47050b63bc4dff9de7c05fe&companyCode=ZE&sippCode=MCMR&pickUpLocation=MUC&dropOffLocation=MUC&pickUpDate=2018-04-02&dropOffDate=2018-04-05&pickUpHour=1200&dropOffHour=1200&ccrc=ZEZDRFSD&rateType=7&pickUpAddress=NA&dropOffAddress=NA&country=DE&source=ES&cdCode=NA&pcCode=NA&tp=1&language=en
```
