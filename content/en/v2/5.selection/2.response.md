Server Response
======================

A separate array is received with the following positions:

```
    "uniqid",
    "carInfo",
    "rates",
    "getData"
    "rateInfo"
    "selectionInfo"
    "cancellationPolicy"
    "currencies"
    "carRentalTerms" // NEW!!!
```

## Values

| VARIABLE   | IMPORTANCE    | DESCRIPTION                                                                                                              |
| ---------- | ------------- | ------------------------------------------------------------------------------------------------------------------------ |
| baseAprox  | `Informative` | Indicates the approximate value of the commissionable value, this is used to determine what the client should be charged |
| taxAprox   | `Informative` | Indicates the approximate value that the customer must pay at the counter                                                |
| totalAprox | `Informative` | Indicates the approximate value of the total rent                                                                        |
| amount     | `Very high`   | Amount to show on page 3, total fee                                                                                      |

## Inclusions parameters

| FIELD       | MANDATORY | MEANING
| ----------- |:---------:| -------:
| description | YES | Text with description of inclusion 
| limit | NO | Return limit Kilometers/Miles included in the rate, when doesnt have limit show 'Unlimited'
| currency | NO | deductible currency
| excess | NO | deductible excess amount
| liability | NO | liability amount to CDW inclusion
| policyName | NO | Car Policy aditional information
