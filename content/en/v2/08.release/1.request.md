Service Parameters
==================

Service to request the activation of a 'onhold' reservation

Service: `release`
-----------------

| URL                                                | ENVIRONMENT  |
| -------------------------------------------------- | ------------ |
| https://api.agentcars.com/v2/sites/release         | production   |
| https://api.dev.agentcars.com/v2/sites/release     | development  |


Request Parameters
---------------------

All parameters are sent with `POST`, explanation of each parameter:

| FIELD            | MANDATORY | MEANING                                              |
| ---------------- | --------- | ---------------------------------------------------- |
| lastName         | YES       | Customer's last name                                 |
| referenceCode    | YES       | Reference code of reservation                        |

Example:
```
  lastName:test
  referenceCode:123123
```
