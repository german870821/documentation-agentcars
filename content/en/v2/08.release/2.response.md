Explanation of each parameter:

| VARIABLE                      | MEANING                                                                                                               |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| confirmationCode              | Booking confirmation number with the rental company                                                                   |
| firstName                     | Customer First Name                                                                                                   |
| lastName                      | Customer last Name                                                                                                    |
| carDescription                | Description of vehicle                                                                                                |
| sippCode                      | Type of vehicle according SIPP code                                                                                   |
| carModel                      | Vehicle Model                                                                                                         |
| rateCode                      | Rate Code                                                                                                             |
| rateTittle                    | Rate Title                                                                                                            |
| rateDescription               | Rate Description                                                                                                      |
| companyImage                  | Picture of vehicle rental company                                                                                     |
| companyName                   | Name of vehicle rental company                                                                                        |
| days                          | Rental Days                                                                                                           |
| carImage                      | Vehicle image                                                                                                         |
| currency                      | Currency                                                                                                              |
| type                          | prepaid (ppd) or pay on destination (pod)                                                                             |
| voucherAmount                 | Amount that should be shown on the voucher as "Voucher Amount". This field is exclusive of prepaid reservations "PPD" |
| voucherNumber                 | Number that should be shown on the voucher as "Voucher Number". This field is exclusive of prepaid reservations "PPD" |
| rateAmount                    | Reservation Value                                                                                                     |
| taxesAmount                   | Taxes Value                                                                                                           |
| doors                         | Number of vehicle doors                                                                                               |
| passengers                    | Max number of passengers in the vehicle                                                                               |
| bags                          | Number of bags which supports the vehicle                                                                             |
| airConditioners               | Air conditioning (Yes/No)                                                                                             |
| transmission                  | Transmission Type (Manual / Automatic)                                                                                |
| pickUpLocation                | based on `IATA` code of destination airport                                                                           |
| dropOffLocation               | based on `IATA` code of return airport                                                                                |
| pickUpAddress                 | Arrival airport customer                                                                                              |
| dropOffAddress                | Return airport customer                                                                                               |
| pickUpDate                    | Pickup Date - Format yyyy-mm-dd                                                                                       |
| dropOffDate                   | Dropoff Date - Format yyyy-mm-dd                                                                                      |
| pickUpHour                    | Pickup Hour - Format militar time example: 0800 -> 8:00 am                                                            |
| dropOffHour                   | Dropoff Hour - Format militar time example: 4 pm => 1600                                                              |
| status                        | Status of reservation                                                                                                 |
| flight_number                 | Flight Number                                                                                                         |
| fFlyerNbr                     | Frequent Flyer Number                                                                                                 |
| fFlyerCarrier                 | Airline code                                                                                                          |
| fmember                       | Member Number                                                                                                         |
| convertedCurrency             |  Currency defined by agency to agentcars return amounts. **                                                           |
| convertedrateAmount           | Reservation amount converted to agency currency defined. **                                                           |
| convertedTaxesAmount          | Tax amount converted to agency currency defined. **                                                                   |
| agency_settlement_currency    | Currency with us for the commission. **                                                                               |
| agency_commission_settlement  | Agency commission with us. **                                                                                         |
| voucherInformation            | Voucher Information - Arrangement with information divided into 3 categories (important, standard, additional)        |
| rateInfo                      | Contains the type of rate inclusion (tax), brings the title, description and inclusions information available of each. More information [here](/v2/selection/response#explanation-of-the-content-of-each-of-the-array-positions).   |

** This field its only returned when agency contact us and request to API response return all amounts with specific currency. 

#### voucherInformation

Arrangement with information divided into 4 categories explained below:

| KEY                   | VALUE                                                                                                                             |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| important             | Important information, mandatory to charge the voucher to be delivered to the customer, it can bring title, description, image and additional information (accountNumber, iata, HCCNumber)
| standard              | Standard information, mandatory to charge the voucher to be delivered to the customer, it can bring title, description and image  |
| additional            | Additional information, optional, it can bring title, description and image                                                       |
| cancellationPolicy    | Cancellation Policy Information, optional                                                                                         |

More information about Voucher [here](/v2/certification-process/info#vouchers-y-confirmations).
