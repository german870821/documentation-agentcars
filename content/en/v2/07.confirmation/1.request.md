Service Parameters
============================

It is responsible for conducting and bookings, according to vehicle data, the rate and the information required to perform it.

Service: `confirmation`
------------------

| URL                                                  | ENVIRONMENT |
| ---------------------------------------------------- | -----------:|
| https://api.agentcars.com/v2/sites/confirmation      | production  |
| https://api.dev.agentcars.com/v2/sites/confirmation  | development |


All parameters are sent with `POST`

| VARIABLE         | MANDATORY      | VALUE                                     | MEANING
| -----------------| -------------- | ----------------------------------------- | ------------------------------------------------------------------------------ |
| uniqid           | YES            | 58e7a97b506d4                             | Session identifier of the transaction in process.                              | 
| paymentType      | YES            | pod                                       | Type of payment of the pod rate: if it is payment at destination, ppd: prepaid | 
| rateType         | YES            | 3                                         | Type of rate                                                                   | 
| companyCode      | YES            | ZT                                        | Company Code from the chosen vehicle                                           | 
| sippCode         | YES            | CCAR                                      | type of vehicle according SIPP code                                            | 
| pickUpLocation   | YES            | LAX                                       | Pickup City * - based on `IATA` code of destination airport                    | 
| dropOffLocation  | YES            | LAX                                       | Dropoff City * - based on `IATA` code of return airport                        | 
| pickUpDate       | YES            | 2022-10-22                                | Pickup Date - Format yyyy-mm-dd                                                | 
| dropOffDate      | YES            | 2022-10-24                                | Dropoff Date - Format yyyy-mm-dd                                               | 
| pickUpHour       | YES            | 1200                                      | Pickup Hour - Format militar time example: 0800 -> 8:00 am                     | 
| dropOffHour      | YES            | 1200                                      | Dropoff Hour - Format militar time example: 0800 -> 8:00 am                    | 
| pickUpAddress    | YES            | LAX, Los Angeles International Airport... | Name of pickup office - It comes in response to the call getSelection          | 
| dropOffAddress   | YES            | LAX, Los Angeles International Airport... | Name of dropoff office - It comes in response to the call getSelection         | 
| firstName        | YES            | test                                      | First Name                                                                     | 
| lastname         | YES            | test                                      | Last Name                                                                      | 
| age              | YES            | 1                                         | Age                                                                            | 
| email            | YES            | test@test.com                             | Email                                                                          | 
| realBase         | YES            | 118.00                                    | Base commissionable                                                            | 
| realTax          | YES            | 20.00                                     | Taxes                                                                          | 
| total            | YES            | 118.00                                    | Amount                                                                         | 
| currency         | YES            | USD                                       | Currency                                                                       | 
| cdCode           | NO             | CDO12                                     | Discount Code                                                                  | 
| pcCode           | NO             | PC1212                                    | Promotional Code                                                               | 
| cbs              | NO             | 1/null                                    | Child Booster Seat                                                             | 
| cst              | NO             | 1/null                                    | Baby Booster Seat                                                              | 
| gps              | NO             | 1/null                                    | GPS                                                                            | 
| skyracks         | NO             | 1/null                                    | skyracks                                                                       | 
| language         | NO             | es/en                                     | ISO To set language response. *                                                | 
| flight_number    | NO             | AA1234                                    | Flight Number                                                                  | 
| fFlyerNbr        | NO             | ZXY987                                    | Frequent Flyer Number                                                          | 
| fFlyerCarrier    | NO             | A3                                        | Airline code, download airline list [here](https://api.dev.agentcars.com/airlines.csv)    | 
| fmember          | NO             | 0123456789                                | Member Number, only for the following companies: Avis, Hertz and National      |
| on_hold          | NO             | 1                                         | Param with value 1 if need to specify that reserve have status 'On Hold'       |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)
