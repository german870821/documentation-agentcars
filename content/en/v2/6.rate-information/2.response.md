Server Response
===============

A separate array is received with the following positions:

    "detail",
    "base",
    "realBase",
    "tax",
    "realTax",
    "realCurrency",
    "additionals",
    "agency_commission",
    "agency_commissionable_value",
    "agency_commission_currency",
    "agency_settlement_currency",
    "agency_commission_settlement",

### Values

| VARIABLE                      | IMPORTANCE  | DESCRIPTION                                                 |
| ----------------------------- | ----------- | ----------------------------------------------------------- |
| **`base`**                    | `Very high` | Base rate without tax                                       |
| realBase                      | `Very high` | Base commissionable                                         |
| total                         | `High`      | Total Approximate of booking, including taxes               |
| tax                           | `Very high` | Taxes to pay with counter                                   |
| realTax                       | `Very high` | Real tax of rate                                            |
| agency_commission             | `High`      | Agency commission in the currency of the reserve            |
| agency_commissionable_value   | `High`      | Commissionable value of the agency in the reserve currency  |
| agency_commission_currency    | `High`      | Reserve currency for commission                             |
| agency_settlement_currency    | `High`      | Currency with us for the commission                         |
| agency_commission_settlement  | `High`      | Agency commission with us                                   |

```
  "base": "20.00",
  "realBase": "20.00",
  "total": "31.92",
  "tax": "11.92",
  "realTax": "11.92",
  "realCurrency": "USD",
  "agency_commission": "2.00",
  "agency_commissionable_value": "20.00",
  "agency_commission_currency": "USD",
  "agency_settlement_currency": "USD",
  "agency_commission_settlement": "2.00"
```

### Additionals

Values to pay at counter, are not included in the costs

```
  "aditionals": {
    "CSI": "8.31/per day",
    "CST": "8.31/per day",
    "GPS": "15.44/per day"
  }
```
