Service Parameters
==================

Calculate the value of the additional fees as inclusions for vehicle rental

Service: `rateInformation`
---------------------

| URL                                                          | ENVIRONMENT |
| ------------------------------------------------------------ | ----------- |
| https://api.agentcars.com/v2/sites/get-rate-information      | production  |
| https://api.dev.agentcars.com/v2/sites/get-rate-information  | development |


Request Parameters
---------------------

All params are sent with `GET`

| VARIABLE         | MANDATORY      | MEANING                                                                                   |
| -----------------| -------------- | ----------------------------------------------------------------------------------------- |
| paymentType      | YES            | Payment option (ppd => Prepaid, pod => Pay on Destination)                                |
| rateType         | YES            | Rate identifier                                                                           |
| uniqid           | YES            | Transaction identifier, recieve inside [get-selection](/v2/selection/response) response   |
| language         | NO             | ISO To set language response *                                                            |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

Sample request via `GET`

```
https://api.dev.agentcars.com/v2/sites/get-rate-information?access-token=72163f79ad5e45256e448e94e6a106bd&paymentType=pod&rateType=1&uniqid=58e7b24f7e5a2&language=en

Parameters sent in the above string:
    paymentType=pod
    rateType=1
    uniqid=58e7b24f7e5a2
    language=en
    
```
