Server Response
===============

In the response you recive an array of category cars,  with a maximum of 7 companies each including PROMO
example:

```
    Economy
    Compact
    Intermediate
    Standard
    Fullsize
    Small SUV
    Minivan
    Standard SUV
    Premium
    Convertible
    Large SUV
    Maxivan
    Super Luxury
```

Within each category are delivered car of each company, ordered from lowest price (Ascending)

### Values

| VARIABLE | IMPORTANCE | DESCRIPTION
| --- | --- | ---:
| baseAprox | `Informative` | Indicates the approximate value of the commissionable value, this is used to determine what the customer should be charged
| taxAprox | `Informative` | Indicates the approximate value that the customer must pay at the counter
| rateConverted | `Informative` | It is the gross rate converted to USD, this serves to order the rates, when they come in multiple currencies, but it is not taken into account
| rateAmount | `Very high` | Amount to be shown in the car comparison matrix

### Messages

Within the information of each car, a "messages" node can come, providing information regarding it:

| VARIABLE | DESCRIPTION
| --- | ---:
| text | The text of the message associated with this cart
| is_offer | 1: If the message is an offer, 0: if it is only informative
