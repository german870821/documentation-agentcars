Service Parameters
==================

It is responsible for sending all available cars per companies with categories, according to a [Rates](rates.md)


Service: `getMatrix`
---------------------

| URL                                                | ENVIRONMENT |
| -------------------------------------------------- | ------------|
| https://api.agentcars.com/v2/sites/get-matrix      | production  |
| https://api.dev.agentcars.com/v2/sites/get-matrix  | development |


Request Parameters
------------------

All params are sent with `GET`

| VARIABLE         | MANDATORY      |MEANING
| ---------------- |:--------------:|--------------------------:
| rateType         | YES            |Type of rate, use `best` or get from [Rates](rates.md)
| country          | YES            |Country Office * country code alpha-2: United States (US), Colombia (CO)
| source           | YES            |Source Country * country code alpha-2: United States (US), Colombia (CO)
| pickUpDate       | YES            |Pickup Date - Format yyyy-mm-dd
| pickUpHour       | YES            |Pickup Hour - Format military time example: 0800 -> 8:00 am, 1600 -> 04:00pm
| dropOffDate      | YES            |Dropoff Date - Format yyyy-mm-dd
| dropOffHour      | YES            |Dropoff Hour - Format military time example: 0800 -> 8:00 am, 1600 -> 04:00pm
| pickUpLocation   | YES            |Pickup City * - based on `IATA` code of destination office/airport
| dropOffLocation  | YES            |Dropoff City * - based on `IATA` code of return office/airport
| pickUpAddress    | YES            |Name of pickup office (Default NA)
| dropOffAddress   | YES            |Name of dropoff office (Default NA)
| cdCode           | NO             |Discount Code 
| pcCode           | NO             |Promotion Code
| companyCode      | NO             |Company Code from the chosen vehicle (if include discount or promotional code)
| language         | NO             |ISO To set language response. *
| paymentType      | NO             |Payment Type of the results shown (ppd: Pay Now, pod: Pay at Destination)
| lat              | YES**          |Pickup Latitude
| lng              | YES**          |Pickup Longitude
| latDropOff       | YES**          |Dropoff Latitude
| lngDropOff       | YES**          |Dropoff Longitude


*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

**Only apply mandatory in case of search outside offices.

### For search availability in Airports:

The airport code `IATA` is a three-letter code designating each airport around the world, these are 3 letters, as follows:

|CODE  |AIRPORT   |
|------|----------|
|MIA   |Miami International Airport
|LAX   |Los Angeles International Airport
|MUC   |Munich Intl Airport

Important: This only responds availability at the selected airports, no offices nearby. The response include all companies available in the airport

Sample request via GET

```
- Example airport search by IATA code:
    https://api.dev.agentcars.com/v2/sites/get-matrix?access-token=7fba8ec9ec1709f76d3cd6495008eabd&pickUpLocation=MUC&pickUpAddress=NA&dropOffLocation=MUC&dropOffAddress=NA&pickUpDate=2020-06-03&dropOffDate=2020-06-05&pickUpHour=1200&dropOffHour=1200&companyCode=NA&cdCode=NA&pcCode=NA&country=DE&source=ES&rateType=best&language=en

    //Parameters sent in the above string:
    pickUpLocation:MUC
    pickUpAddress:NA
    dropOffLocation:MUC
    dropOffAddress:NA
    pickUpDate:2020-06-03
    dropOffDate:2020-06-05
    pickUpHour:1200
    dropOffHour:1200
    companyCode:NA
    country:DE
    source:ES
    rateType:best
    language:en
    paymentType:ppd
```

### For specific offices:

`IATA` codes for offices are 6 letters, as follows:

|CODE    |OFFICE    |
|--------|----------|
|MIAE08  |HERTZ, MIAMI BEACH, FL 33139
|MIAC72  |ALAMO, 100 CHOPIN PLZ, FL 33131-4323

Important: This responds availability at the selected office and offices nearby.

```
- Example specific offices:
    https://api.dev.agentcars.com/v2/sites/get-matrix?access-token=7fba8ec9ec1709f76d3cd6495008eabd&pickUpLocation=MIAE08&pickUpAddress=NA&dropOffLocation=MIAC72&dropOffAddress=NA&pickUpDate=2020-06-03&dropOffDate=2020-06-05&pickUpHour=1200&dropOffHour=1200&companyCode=NA&cdCode=NA&pcCode=NA&country=US&source=AR&rateType=best&language=en

    //Parameters sent in the above string:
    pickUpLocation:MIAE08
    pickUpAddress:NA
    dropOffLocation:MIAC72
    dropOffAddress:NA
    pickUpDate:2020-06-03
    dropOffDate:2020-06-05
    pickUpHour:1200
    dropOffHour:1200
    companyCode:NA
    country:US
    source:AR
    rateType:best
    language:en
    paymentType:ppd
```

### For offices outside airport per Lat, Lng:

Set `City` as pickUpLocation and dropOffLocation if are the same, if are different set `City` in pickUpLocation and `City2` in dropOffLocation

Important: This responds availability 5 miles around ordered by lowest price, so can be exist a more close office but not need to be the first one that appear or maybe don't appear because it's closed at the selected time. Also can have different offices from the same company.

Sample request via GET

```
   
- Example for searches outside offices:
    https://api.dev.agentcars.com/v2/sites/get-matrix?access-token=7fba8ec9ec1709f76d3cd6495008eabd&pickUpLocation=City&pickUpAddress=NA&dropOffLocation=City&dropOffAddress=NA&pickUpDate=2020-06-05&dropOffDate=2020-06-12&pickUpHour=1200&dropOffHour=1200&companyCode=NA&cdCode=NA&pcCode=NA&country=US&source=CO&rateType=1&lat=25.77427&lng=-80.19366&latDropOff=25.77427&lngDropOff=-80.19366
    
    //Parameters sent in the above string:
    pickUpLocation:City
    pickUpAddress:NA
    dropOffLocation:City
    dropOffAddress:NA
    pickUpDate:2020-06-05
    dropOffDate:2020-06-12
    pickUpHour:1200
    dropOffHour:1200
    companyCode:NA
    cdCode:NA
    pcCode:NA
    country:US
    source:CO
    rateType:best
    lat:25.77427
    lng:-80.19366
    latDropOff:25.77427
    lngDropOff:-80.19366

-Example Pickup Location different that Dropoff Location, example: Miami Beach to Fort Lauderdale

    https://api.dev.agentcars.com/v2/sites/get-matrix?access-token=7fba8ec9ec1709f76d3cd6495008eabd&pickUpLocation=City&pickUpAddress=NA&dropOffLocation=City2&dropOffAddress=NA&pickUpDate=2020-06-05&dropOffDate=2020-06-12&pickUpHour=1200&dropOffHour=1200&companyCode=NA&cdCode=NA&pcCode=NA&country=US&source=CO&rateType=best&lat=25.79065&lng=-80.13005&latDropOff=26.12231&lngDropOff=-80.14338

    //Parameters sent in the above string:
    pickUpLocation:City
    pickUpAddress:NA
    dropOffLocation:City2
    dropOffAddress:NA
    pickUpDate:2020-06-05
    dropOffDate:2020-06-12
    pickUpHour:1200
    dropOffHour:1200
    companyCode:NA
    cdCode:NA
    pcCode:NA
    country:US
    source:CO
    rateType:best
    lat:25.79065
    lng:-80.13005
    latDropOff:26.12231
    lngDropOff:-80.14338

```
