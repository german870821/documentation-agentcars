Service Parameters
============================

See details of a reservation

Service: `my-reservation`
---------------------

| URL                                                    | ENVIRONMENT |
| ------------------------------------------------------ | ----------- |
| https://api.agentcars.com/v2/sites/my-reservation      | production  |
| https://api.dev.agentcars.com/v2/sites/my-reservation  | development |


Request Parameters
---------------------

All parameters are sent with `POST`, explanation of each parameter:

| FIELD             | MANDATORY | MEANING                                               |
| ----------------- | --------- | ----------------------------------------------------- |
| lastName          | YES       | Customer's last name                                  |
| confirmationCode  | YES       | Booking confirmation number with the rental company   |
| language          | NO        | ISO To set language response *                        |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

Example:
```
  lastName:test
  confirmationCode:G4N410
  language:en
```
