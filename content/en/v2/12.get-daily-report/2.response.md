Server Response
===============    

A separate array is received with all daily reservation from user (or agency if user its supervisor or administrator) :

```
        {
            "confirmationCode",
            "firstName",
            "lastName",
            "email",
            "pickUpDate",
            "dropOffDate",
            "date",
            "status",
            "paymentType"
        },
```
