Explanation of each parameter:

| VARIABLE                     | IMPORTANCE  | MEANING                                                                                                      |
|------------------------------|-------------|------------------------------------------------------------------------------------------------------------- |
| confirmationCode             | `High`      | Booking confirmation number with the rental company                                                          |
| firstName                    | `High`      | Customer First Name                                                                                          |
| lastName                     | `High`      | Customer last Name                                                                                           |
| carDescription               | `High`      | Description of vehicle                                                                                       |
| sippCode                     | `High`      | Type of vehicle according SIPP code                                                                          |
| carModel                     | `High`      | Vehicle Model                                                                                                |
| rateCode                     | `High`      | Rate Code                                                                                                    |
| rateTittle                   | `High`      | Rate Title                                                                                                   |
| rateDescription              | `High`      | Rate Description                                                                                             |
| companyImage                 | `High`      | Picture of vehicle rental company                                                                            |
| companyName                  | `High`      | Name of vehicle rental company                                                                               |
| days                         | `High`      | Rental Days                                                                                                  |
| carImage                     | `High`      | Vehicle image                                                                                                |
| currency                     | `High`      | Currency                                                                                                     |
| type                         | `High`      | prepaid (ppd) or pay on destination (pod)                                                                    |
| voucherAmount                | `Very high` | Amount that should be shown on the voucher as "Voucher Amount". This field is exclusive of prepaid reservations "PPD"    |
| voucherNumber                | `Very high` | Number that should be shown on the voucher as "Voucher Number". This field is exclusive of prepaid reservations "PPD"    |
| rateAmount                   | `High`      | Reservation Value                                                                                            |
| taxesAmount                  | `High`      | Taxes Value                                                                                                  |
| doors                        | `High`      | Number of vehicle doors                                                                                      |
| passengers                   | `High`      | Max number of passengers in the vehicle                                                                      |
| bags                         | `High`      | Number of bags which supports the vehicle                                                                    |
| airConditioners              | `High`      | Air conditioning (Yes/No)                                                                                    |
| transmission                 | `High`      | Transmission Type (Manual / Automatic)                                                                       |
| pickUpLocation               | `High`      | based on `IATA` code of destination airport                                                                  |
| dropOffLocation              | `High`      | based on `IATA` code of return airport                                                                       |
| pickUpAddress                | `High`      | Arrival airport customer                                                                                     |
| dropOffAddress               | `High`      | Return airport customer                                                                                      |
| pickUpDate                   | `High`      | Pickup Date - Format yyyy-mm-dd                                                                              |
| dropOffDate                  | `High`      | Dropoff Date - Format yyyy-mm-dd                                                                             |
| pickUpHour                   | `High`      | Pickup Hour - Format militar time example: 0800 -> 8:00 am                                                   |
| dropOffHour                  | `High`      | Dropoff Hour - Format militar time example: 4 pm => 1600                                                     |
| status                       | `High`      | Status of reservation *                                                                                      |
| flight_number                | `High`      | Flight Number                                                                                                |
| fFlyerNbr                    | `High`      | Frequent Flyer Number                                                                                        |
| fFlyerCarrier                | `High`      | Airline code                                                                                                 |
| fmember                      | `High`      | Member Number                                                                                                |
| convertedCurrency            | `High`      | Currency defined by agency to agentcars return amounts. **                                                   |
| convertedrateAmount          | `High`      | Reservation amount converted to agency currency defined. **                                                  |
| convertedTaxesAmount         | `High`      | Tax amount converted to agency currency defined. **                                                          |
| agency_settlement_currency   | `High`      | Currency with us for the commission. **                                                                      |
| agency_commission_settlement | `High`      | Agency commission with us. **                                                                                |
| voucherInformation           | `Very high` | Voucher Information - Arrangement with information divided into 3 categories (important, standard, additional)|
| rateInfo                     | `Very high` | Contains the type of rate inclusion (tax), brings the title, description and inclusions information available of each. More information [here](getSelection-response.md#explanation-of-the-content-of-each-of-the-array-positions). |

*For the process to follow when this field comes with the value "On Request", please see the section [Reservations with Status On Request](reservations-onRequest.md).

** This field its only returned when agency contact us and request to API response return all amounts with specific currency. 

#### voucherInformation

Arrangement with information divided into 4 categories explained below:

| KEY                | VALUE                                                                                                                            |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------- |
| important          | Important information, mandatory to charge the voucher to be delivered to the customer, it can bring title, description, image and additional information (accountNumber, iata, HCCNumber)   |
| standard           | Standard information, mandatory to charge the voucher to be delivered to the customer, it can bring title, description and image |
| additional         | Additional information, optional, it can bring title, description and image                                                      |
| cancellationPolicy | Cancellation Policy Information, optional                                                                                        |

More information about Voucher [here](certification-process.md#vouchers-y-confirmations).
