Rates - Request parameters `REST`
=================================

Service request the availables rates to specific source-destination

Service
-------
| URL                                                    | ENVIRONMENT |
| ------------------------------------------------------ | ------------|
| https://api.agentcars.com/v2/sites/rates               | production  |
| https://api.dev.agentcars.com/v2/sites/rates           | development |



Request Parameters
---------------------

All parameters are sent with `GET`, explanation of each parameter:

| FIELD     | MANDATORY | MEANING                                   |
| --------- | --------- | ----------------------------------------- |
| country   | YES       | Country code of destination to rent a car |
| source    | YES       | Country code of source of customer        |
| language  | NO        | ISO To set language response *            |

 *Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

---
 
```
    - Example:
        https://api.dev.agentcars.com/v2/sites/rates?access-token=4824a8dc7626f7b5a5338591abe588a2&country=US&source=CO&language=en
    
    - Parameters on request:
        country:US
        source:CO
        language=en
```
