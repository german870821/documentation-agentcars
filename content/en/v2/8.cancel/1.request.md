Service Parameters
==================

Service to request the cancellation of a reservation

Service: `cancel`
-----------------
| URL                                                | ENVIRONMENT  |
| -------------------------------------------------- | ------------ |
| https://api.agentcars.com/v2/sites/cancel          | production   |
| https://api.dev.agentcars.com/v2/sites/cancel      | development  |


Request Parameters
---------------------

All parameters are sent with `POST`, explanation of each parameter:

| FIELD            | MANDATORY | MEANING                                              |
| ---------------- | --------- | ---------------------------------------------------- |
| lastName         | YES       | Customer's last name                                 |
| confirmationCode | YES       | Booking confirmation number with the rental company  |

Example:
```
  lastName:test
  confirmationCode:G4N410
```
