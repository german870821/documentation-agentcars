Get URL 
-------------------

Request to get search URL according data sent:

| URL                                                        | ENVIROMENT    |
| ---------------------------------------------------------- | ------------- |
| https://cars.agencydomain.com/en/site/get-url/             | production    |
| https://www.agentcars.com/subsite/en/site/get-url/         | development   |

| DATA                 | MEANING                      |
| -------------------- | ---------------------------- |
| en                   | Language*                    |
| cars                 | Agency subdomain             |
| agency-domain.com    | Agency URL                   |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

Search request parameters
-------------------

All params are sent with `GET`

| FIELD            | MANDATORY     | MEANING                                                                    |
| ---------------- | ------------- | -------------------------------------------------------------------------- |
| pickUpLocation   | YES           | `IATA` code of pickup location                                             |
| dropOffLocation  | YES           | `IATA` code of dropoff location                                            |
| pickUpDate       | YES           | Pickup Date - Format yyyy-mm-dd                                            |
| pickUpHour       | YES           | Pickup Hour - Format militar time example: 0800 -> 8:00 am                 |
| dropOffDate      | YES           | Dropoff Date - Format yyyy-mm-dd                                           |
| dropOffHour      | YES           | Dropoff Hour - Format militar time example: 0800 -> 8:00 am                |
| source           | YES           | Source Country * country code alpha-2: United States (US), Colombia (CO)   |
| country          | YES           | Country Office * country code alpha-2: United States (US), Colombia (CO)   |
| paymentType      | NO            | Payment Type of the results shown (ppd: Pay Now, pod: Pay at Destination)  |
| lang             | NO            | Language to return data. *                                                 |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

### pickUpLocation and dropOffLocation: IATA

The `IATA` airport code is a three letter code that designates each airport in the world, these are 3 letters, as follows:

| CODE   | AIRPORT                            |
| ------ | ---------------------------------- |
| MIA    | Miami International Airport        |
| LAX    | Los Angeles International Airport  |

### Example of request

#### Pick-up airport equal to drop-off airport

```
Production
https://cars.agencydomain.com/es/site/get-url/?pickUpDate=2019-08-21&pickUpHour=1200&dropOffDate=2019-08-24&dropOffHour=1200&pickUpLocation=MIA&dropOffLocation=MIA

Test
https://www.agentcars.com/subsite/es/site/get-url/?pickUpDate=2019-08-21&pickUpHour=1200&dropOffDate=2019-08-24&dropOffHour=1200&pickUpLocation=MIA&dropOffLocation=MIA

//Parameters sent in the above string:
pickUpLocation:MIA
dropOffLocation:MIA
pickUpDate:2019-08-21
dropOffDate:2019-08-24
pickUpHour:1200
dropOffHour:1200
```
