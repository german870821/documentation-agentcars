URL Suggest
-------------------

Send a request to next url to obtain airports and cities by text sent:

| URL                                                        | ENVIROMENT    |
| ---------------------------------------------------------- | ------------- |
| https://suggest.agentcars.com/suggest/                     | production    |

Request Parameters
-------------------

Todos los parámetros de envíos van por `GET`

| FIELD            | MANDATORY     | MEANING                                          |
| ---------------- | ------------- | ------------------------------------------------ |
| lang             | YES           | language to return a response with ISO format. * |
| query            | YES           | Text to search (ex: Miami)                       |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

Response Parameters
-------------------

| FIELD            | MEANING                                                                      |
| ---------------- | ---------------------------------------------------------------------------- |
| airport          | `true` if its airport, `false` if not.                                       |
| cityLoc          | `true` if its city, `false`if not.                                           |
| countryCode      | Country of location, with Alpha2 code. Ex: United States (US), Colombia (CO) |
| hasoffice        | Number of nearby offices.                                                    |
| iata             | `IATA` Code of location (if have)                                            |
| latitude         | Location latitude.                                                           |
| longitude        | Location longitude.                                                          |
| schedule         | Location schedule by days.                                                   |
| timezone         | Time Zone (ex: America/New_York).                                            |
| value            | Location name, with format (NAME, IATA, STATE, COUNTY). Ex: Miami Intl Airport, MIA, Florida, United States of America.  |

#### Schedule: `schedule`

Schedule from location, separate by days (1: Monday, 2: Thursday, 3: Wednesday, 4: Tuesday, 5: Friday, 6: Saturday, 7: Sunday).

How to build the Search URL
-------------------

| URL                                                        | ENVIROMENT   |
| ---------------------------------------------------------- | ------------ |
| https://cars.agency-domain.com/en/site/search/             | production   |
| https://www.agentcars.com/subsite/en/site/search/          | development  |


| DATA                | MEANING                           |
| ------------------- | --------------------------------- |
| en                  | Language*                         |
| cars                | Agency subdomain                  |
| agency-domain.com   | Agency URL                        |

*Available Languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

Request Parameters on search
-------------------

All params are sent with `GET`

| FIELD            | MANDATORY     | MEANING
| ---------------- | ------------- | -------------------------------------------------------------------------- |
| rateType         | YES           | Constant value: best                                                       |
| pickUpLocation   | YES           | `IATA` code of pickup location                                             |
| dropOffLocation  | YES           | `IATA` code of dropoff location                                            |
| pickUpDate       | YES           | Pickup Date - Format yyyy-mm-dd                                            |
| pickUpHour       | YES           | Pickup Hour - Format militar time example: 0800 -> 8:00 am                 |
| dropOffDate      | YES           | Dropoff Date - Format yyyy-mm-dd                                           |
| dropOffHour      | YES           | Dropoff Hour - Format militar time example: 0800 -> 8:00 am                |
| source           | YES           | Source Country * country code alpha-2: United States (US), Colombia (CO)   |
| country          | YES           | Country Office * country code alpha-2: United States (US), Colombia (CO)   |
| paymentType      | NO            | Payment Type of the results shown (ppd: Pay Now, pod: Pay at Destination)  |
| lat              | YES**         | Pickup Latitude                                                            |
| lng              | YES**         | Pickup Longitude                                                           |
| latDropOff       | YES**         | Dropoff Latitude. If return car in other city.                             |
| lngDropOff       | YES**         | Dropoff Longitude. If return car in other city.                            |

*Available languages: English (en), Spanish (es), Japanese (ja), Korean (ko), Portuguese (pt), German (de), French (fr), Italian (it), Arabic (ar)

**Only apply mandatory with search by cities

### IATA

#### To Airports

The `IATA` airport code is a three letter code that designates each airport in the world, these are 3 letters, as follows:

| CODE    | AIRPORT                           |
| ------  | --------------------------------- |
| MIA     | Miami International Airport       |
| LAX     | Los Angeles International Airport |

#### To cities

Not included, you must to send next information according to search to request (mandatory include latitude and longitude of the city):

|CODE  |MEANING   |
|--------|------------|
|City    |On pickUpLocation and dropOffLocation, If the location to pick and return the car its the same.
|City2   |Change dropOffLocation whit this parameter, if going to return the car in another city. Must to include latitude and longitude from tho cities. 



### Example of request

#### To Airports Example Request

```
Development
https://www.agentcars.com/subsite/en/site/search/?rateType=best&pickUpDate=2019-08-21&pickUpHour=1200&dropOffDate=2019-08-24&dropOffHour=1200&pickUpLocation=MIA&dropOffLocation=MIA

//Parámetros enviados en la cadena anterior:
rateType:best
pickUpLocation:MIA
dropOffLocation:MIA
pickUpDate:2019-08-21
dropOffDate:2019-08-24
pickUpHour:1200
dropOffHour:1200
```
