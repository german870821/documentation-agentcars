Server Response
===============

A separate array is received with all offices found withing each schedule, withing each office find next structure :

```
    {
        "lng",
        "lat":,
        "address",
        "address_jp",
        "address_ar",
        "address_ko",
        "address_number",
        "zip_code",
        "city_name",
        "office_code",
        "state",
        "company_name",
        "company_code",
        "country_code",
        "schedule": {
            "1": {
                "close",
                "opening"
            },
            .
            .
            .
            "7": {
                "close",
                "opening"
            }
        },
        "distance"
    },
```


Schedule contain array with days (start monday with code 1 and end sunday with code 7) and opening hours.

```
    "1": {
        "close": "2000",
        "opening": "0800"
    },
    "2": {
        "close": "2000",
        "opening": "0800"
    },
    "3": {
        "close": "2000",
        "opening": "0800"
    },
    "4": {
        "close": "2000",
        "opening": "0800"
    },
    "5": {
        "close": "2000",
        "opening": "0800"
    },
    "6": {
        "close": "1800",
        "opening": "0800"
    },
    "7": {
        "close": "1730",
        "opening": "0900"
    }
```

If some day office not open, system not return day schedule, Ex: office not open Monday and Friday (code 1 and 5):
