Service parameters
================================

Service request the availables offices from lat/lng OR cityCode

Service: `findOffices`
-------
| URL                                                       | ENVIRONMENT  |
| --------------------------------------------------------- | -------------|
| https://api.agentcars.com/v2/sites/find-offices           | production   |
| https://api.dev.agentcars.com/v2/sites/find-offices       | development  |

Request Parameters
---------------------

All parameters are sent with `GET`, explanation of each parameter:

| FIELD       | MANDATORY | MEANING                                                                         |
| ----------- | ----------| ------------------------------------------------------------------------------- |
| distance    | YES       | Distance of miles to find offices around of location requested                  |
| source      | YES       | Country Office * country code alpha-2: United States (US), Colombia (CO)        |
| lat         | YES*      | Latitude                                                                        |
| lng         | YES*      | Longitude                                                                       |
| cityCode    | YES**     | City Code (ex: MIA to Miami)                                                    |
| companyCode | NO        | Company Code from the chosen vehicle (if want offices from specific company)    |

\* Only apply mandatory if send coordinates.

** Only apply mandatory if not send coordinates. 



```
- Example with coordinates:
    https://api.dev.agentcars.com/v2/sites/find-offices?access-token=989e82a2c47050b63bc4dff9de7c05fe&distance=2&source=CO&lat=45.463689&lng=9.1883

    Parameters on request:
        distance:2
        source:CO
        lat:45.463689
        lng:9.1883
        
- Example with cityCode:
    https://api.dev.agentcars.com/v2/sites/find-offices?access-token=989e82a2c47050b63bc4dff9de7c05fe&distance=2&source=CO&cityCode=MIA

    Parameters on request:
        distance:2
        source:CO
        cityCode;MIA
```


This service is called to show offices, for example on a map, according to the parameters sent.

Then the getMatrix service is called once with the corresponding sending parameters, if coordinates are used call this service sending those parameters. The list of offices obtained may not correspond to the list of cars of getMatrix, for different reasons such as schedules, dates, among others.

The detail of sending parameters is found here: [get-matrix](/v2/matrix/request)
